# 🌟 星星学院 · 四年级闯关乐园

一款给**马来西亚华小（SJKC）四年级**学生的趣味学习游戏，依据 **KSSR 课程标准**编写。

- 📱 **随处可玩**：单个 `index.html`，用浏览器打开即可在 iPad、iPhone、电脑上游玩。
- 🔌 **离线运行**：无需联网、无需安装。
- 📖 **学一学**：每个单元先有图文讲解卡帮你弄懂概念，再去闯关。
- 🎮 **游戏化**：选科目 → 学一学 → 闯单元 → 答题得星星积分，连对有奖励、满分撒花。
- 📚 **重在理解**：每题都附**详细讲解**和**记忆小提示**，不只是对错。
- 🧠 **更有挑战**：题目和选项每次随机打乱（不能背答案），答错扣星星，满分才有 3 星。
- 📕 **错题本**：答错的题会自动收进错题本，可随时点击复习；答对就移出错题本，复习不扣星——专门帮你把不会的题变成会做的题。
- 🏆 **成就徽章**：10 枚徽章给你长期目标（满分高手、连对达人、科目通、零错题、游戏全能……），越往后越难拿，解锁时有祝贺动画和音效。
- ♻️ **一键重置**：主页可重置星星、积分与单元锁定（错题本一并清空），方便重新挑战；年级和音效设置会保留。
- 🌠 **奖励小游戏（5 款轮换）**：达成成就解锁奖励关——「宇宙星空漫游」「数字陨石」「飞机大战 ✈️」「坦克大战 🚜」「知识配对 🃏」随机轮换，每次玩法不同；按分数/配对换星、更具挑战，有声音音乐。
- 🧩 **三/四年级 · 多科目**：数学、华文、科学、国语(BM)、English、历史，共 36 单元 114 题（持续扩充中，含两步/三步应用题等高阶题，部分单元附「例题」范例讲解）。

## 使用方法
1. 下载 `index.html`（或从 Google Drive 的 `PrimaryApp` 文件夹获取）。
2. 用 Safari / Chrome 等浏览器打开。
3. 在 iPad 上可用 Safari「分享 → 添加到主屏幕」，像 App 一样使用。
