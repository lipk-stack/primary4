# 星星学院 — 开发进度 (Development Progress)

马来西亚华小（SJKC）**三/四年级** KSSR 学习游戏 · 单文件 · 离线可玩 · iPad/iPhone/电脑通用。

## 目标 (Goals)
- 趣味闯关游戏形式，吸引小学生主动学习。
- 严格遵照马来西亚华小 KSSR 课程标准（三年级 & 四年级），语言用词正确。
- 每题附**详细讲解**与**小提示**，帮助学生真正理解知识点。
- 完全单文件 `index.html`，无需安装、无网络也能玩；遵循 YAGNI 原则（纯原生 JS，无框架/构建）。
- 每次迭代输出到 Google Drive 的 `PrimaryApp` 文件夹分发。

### Iteration 3 — 2026-06-21 — 教学模式 + 防刷随机化 + 第二款奖励游戏
- **「学一学」教学模式（核心增强）：**
  - 点击单元后**先进入图文教学卡**，看懂讲解再开始闯关。
  - 为**全部 36 个单元**各写一张教学卡：`导入说明 + 2~3 条知识要点（编号卡片）+ 黄色例子框`。
  - 华文/数学/科学用中文讲解；BM/English 单元用对应语言（马来文/英文）讲解，语言地道。
  - 没有教学卡的单元会自动跳过直接闯关（向后兼容）。
- **题目与选项双重随机化（让星星更难拿、杜绝背答案）：**
  - 每次开始挑战时打乱**题目顺序**。
  - 同时打乱每道题的**选项顺序**，并自动重新定位正确答案下标（`shuffleOptions`）。
  - 学生无法靠记“第几个选项是对的”来过关，必须真正理解。
- **第二款奖励小游戏「⚡闪电心算」（Canvas）：**
  - 屏幕上方随机出现算式（按年级难度生成：三年级 +/−/× 较小，四年级范围更大）。
  - 数字泡泡从下往上飘动，点中**正确答案**泡泡 +1 星，点错 −1 分（最低 0），并即时换新题。
  - 配独立音效（答对闪光、答错锯齿、开始/结束号角）。
  - **奖励触发时在两款游戏间随机抽取**（🛸宇宙星空漫游 / ⚡闪电心算），真正做到“每次都不一样”。
- 结算页奖励卡文案更新为「随机奖励小游戏」，营造抽奖惊喜感。
- **验证**：Node 脚本校验——36/36 单元教学卡齐全且结构完整、选项无重复（保证答案重定位正确）、答案下标全部合法、JS 语法通过、所有 `$()` 引用的元素 ID 均存在。

## 技术架构 (Architecture)
- 纯单文件 `index.html` — HTML + CSS + Vanilla JS，无框架，无构建步骤。
- `SYLLABUS` 结构：`{ std3/std4: { subjects: { id: { topics: { id: { qs: [] } } } } } }`
- `LESSONS` 结构：`{ "grade.subject.topic": { intro, points:[], eg } }`，与 SYLLABUS 平行，便于续写教学卡。
- 流程：选科目 → 选单元 → **学一学(lesson)** → 答题(随机) → 讲解 → 结算 → 随机奖励游戏。
- `save` 对象（localStorage key: `xingxing_v2`）：`{ stars, points, grade, completed, best, muted, musicMuted }`
- 两款 Canvas 迷你游戏（`game-screen` 宇宙漫游 / `math-screen` 闪电心算）完全自包含于 `<script>`，零依赖。
- `shuffle` / `shuffleOptions`：开局打乱题序与选项并重定位答案下标，防背答案。
