# 星星学院 — 开发进度 (Development Progress)

马来西亚华小（SJKC）**三/四年级** KSSR 学习游戏 · 单文件 · 离线可玩 · iPad/iPhone/电脑通用。

## 目标 (Goals)
- 趣味闯关游戏形式，吸引小学生主动学习。
- 严格遵照马来西亚华小 KSSR 课程标准（三年级 & 四年级），语言用词正确。
- 每题附**详细讲解**与**小提示**，帮助学生真正理解知识点。
- 完全单文件 `index.html`，无需安装、无网络也能玩；遵循 YAGNI 原则（纯原生 JS，无框架/构建）。
- 每次迭代输出到 Google Drive 的 `PrimaryApp` 文件夹分发。

## 已完成 (Done)

### Iteration 3 — 2026-06-22 — 学一学讲解模式 + 防背题 + 第二款奖励游戏
- **「学一学」课前讲解模式（本轮头号功能）**：
  - 新增 `LEARN` 教学库，**覆盖全部 36 个单元**，每单元 1~3 张图文讲解卡。
  - 流程改为：选单元 → **先学一学（讲概念）** → 再开始闯关。直接服务"帮助学生真正理解知识点"。
  - 缺省容错：若某单元暂无讲解则直接进入闯关（YAGNI）。
- **防背题机制（更难拿星）**：
  - 每次闯关**随机打乱题目顺序**。
  - 每题**选项顺序也随机打乱**，并重新定位正确答案 —— 记住"第 2 个是答案"无效。
- **更高难度的星级评定**：满分=3★，≥80%=2★，≥60%=1★，否则 0★（原为 60% 就给 2★）。
- **第二款奖励小游戏「数字陨石」🌠**：
  - 顶部出现算式（按年级生成：三年级乘法表/加法；四年级两位数乘法、加法、整除除法）。
  - 移动熊猫接住**答案正确**的数字陨石 +1 星并换新题；接错数字或碰 ☄️ 扣 1 星。
  - 奖励关触发时**在两款游戏间随机选择**（星空漫游 / 数字陨石），玩法每次不同。复用 Canvas、计时器、音效引擎。
- **奖励关卡升级（4 款游戏轮换，每次不同）**：
  - 新增经典 **「飞机大战 ✈️」**（自动开火击落外星战机、躲避敌方子弹）与 **「坦克大战 🚜」**（炮弹直射、横向移动的敌坦克要算提前量并躲炮火）。
  - 奖励关现在在 **4 款游戏间随机轮换**（太空漫游 / 数字陨石 / 飞机大战 / 坦克大战），并**避免连续两次相同**。
  - **更难拿星**：所有奖励游戏改为 **按分数换星**（捕捉类每 3 分、射击类每 4 分换 1 颗星，最多 8 颗），不再 1 抓 1 星。
  - 射击游戏含耐打敌人（需 2 击）、每局随机速度、敌方火力，受击闪红扣分；新增激光/爆炸音效。
- **扩充题库**：新增多道两步应用题（乘法购书、分数加法等），题量 79 → **85 题**。
- **节点验证**：全题库 85 题答案索引校验通过；36 个 LEARN 键全部对应真实单元；JS 语法校验通过。

## 技术架构 (Architecture)
- 纯单文件 `index.html` — HTML + CSS + Vanilla JS，无框架，无构建步骤。
- `SYLLABUS` 结构：`{ std3/std4: { subjects: { id: { topics: { id: { qs: [] } } } } } }`
- `save` 对象（localStorage key: `xingxing_v2`）：`{ stars, points, grade, completed, best, muted, musicMuted }`
- Canvas 迷你游戏完全自包含于 `<script>` 标签内，零依赖。

## 文件结构 (Files)
- `index.html` — 完整游戏（唯一运行文件）。
- `PROGRESS.md` — 本进度文档，每次迭代续写。
- `README.md` — 项目简介与使用说明。
